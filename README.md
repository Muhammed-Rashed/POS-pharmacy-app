# Pharmacy Point of Sale (POS) System

A cross-platform POS application built for pharmacies, featuring full offline support with cloud synchronization.

## Tech Stack

- **Flutter** UI
- **Firebase Firestore** Cloud sync & real-time updates
- **SQLite** Local offline storage
- **Docker** Containerized deployment & testing

## Features

- **100% Offline Functionality**: Works without an internet connection; data syncs to the cloud when back online
- **Multi-tab Support**: Serve up to 10 customers simultaneously
- **Conflict Resolution**: Handles sync conflicts for unsynced sales and stock inconsistencies
- **Real-time Inventory**: Stock levels update across all devices instantly
- **Cross-platform**: Runs on multiple platforms thanks to Flutter and Docker-tested deployment
